# Бетон через казан

На нашому сервері можна миттєво перетворювати **порошок бетону** у **бетон**, просто **кинувши його в казан із водою**. Жодних зайвих маніпуляцій з джерелами води.

### Як користуватися

1. Постав **казан** і **наповни його водою**.
2. **Кинь** у казан порошок бетону (будь-якого кольору).
3. Забери **готовий бетон**, що випаде предметом.
4. Повторюй порціями (щоб не засмічувати підлогу предметами).

### Швидкий старт будівельника

* Зроби “бетонний цех”: **казан + лійка + скриня** під місцем падіння предметів.
* Тримай **окремі скрині/рамки** під кожний колір — менше плутанини.
* Для великих обсягів кидай по **½ стека або стеку** за раз.

### Поради

* **Лійка** автоматично збере готові блоки — нічого не загубиться.
* **Познач** казан табличкою/рамкою з зразком кольору.
* Для виїзних робіт: **казан + відро води + стеки порошку** = мобільна станція.
* Якщо поруч гравці, **кидай порціями**, щоб не створювати дощ із предметів.

### FAQ

**Працює з усіма кольорами?** Так.\
**Потрібні команди чи меню?** Ні. **Вода в казані + порошок → бетон.**\
**Де краще робити?** У власній майстерні або на спеціально відведених станціях.\
**Чому предмети інколи розлітаються?** Кидай порціями та використовуй **лійку/скриню** для збору.
